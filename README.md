**Pottibl™ is a no-BS, easy and lightweight ZIP-to-USB app to get your files from your laptop straight to your USB drive.** This takes LONG doing it manually, especially if you're not used to computers and laptops. But with Pottibl™ and its custom PortableUI, this makes stuff so easy and bypasses all of those in 2 or even **1 click** and everything automatically does it for you. ***Transferring files were never this easy.***

---

# The Pottibl™ UI - PortableUI 

Pottibl™ runs on its user interface named **PortableUI**. **PortableUI** is a simplistic and minimalist UI for Pottibl™ to make the ZIP-to-USB process easier.

# How Pottibl™ Works

**Pottibl™ works by literally:**
1. Zipping the folder
2. Transferring the ZIP to your USB drive

*all in order.* **That's all it does.**

# How To Download

To download Pottibl™, go to the Releases tab.
Then look for the version with the "latest" tag next to the name.

# How To Use

Once its downloaded, follow these steps in order to start using the application:

1. Right-click on the downloaded file Pottibl.exe
2. Press Run as administrator
3. Then, if a prompt pops up, press Yes
4. Then, if a blue or red screen from Windows SmartScreen pops up, just press More Details ⌄ and press Install anyway or Install
5. Then, drag and drop your folder into the app window
6. It will show several windows popup with progress bars. During this process, make sure your laptop doesn't shutdown or hibernate
7. After that, you will see the success message. Your file is officially moved to your USB drive!
